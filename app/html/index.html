Welcome to the GWI Asset Management System API

This powerful system enables efficient management of various asset types including Charts, Insights, and Audiences. Users can seamlessly add these assets to their favorites for quick and easy access.

API Endpoints

  • GET /ping - Health check endpoint
  • POST /assets/add - Add a new asset (Chart, Insight, or Audience)
  • GET /assets/get - Retrieve a specific asset by ID and type
  • PUT /assets/modify - Modify an existing asset
  • DELETE /assets/delete - Delete an asset
  • POST /user/add - Add a new user to the system
  • POST /user/favorite/add - Add an asset to a user's favorites
  • GET /user/favorites - Get a list of a user's favorite assets

Asset Types

  • Chart: Graphical representation of data with title, axis labels, and data points
  • Insight: Textual information providing valuable market insights
  • Audience: Demographic information about a specific group

Key Features

Concurrent processing of user favorites across multiple shards for improved performance
Secure user management system with authentication
Efficient data storage and retrieval using MySQL database
RESTful API for easy integration with front-end applications
Comprehensive logging and error handling

Getting Started

To start using the GWI Asset Management System API:

  1. Register a new user using the POST /user/add endpoint
  2. Add assets using the POST /assets/add endpoint
  3. Add favorites for a user with POST /user/favorite/add
  4. Retrieve user favorites using GET /user/favorites or GET /user/favorites/detailed

For more detailed information on how to use these endpoints, please refer to the API documentation.
